Class CollisionPrimitive

CollisionPrimitives are the basic building blocks for collision objects. They are defined by a primitive type, its size and pose.

Parameters

Gets the parameter array of the current CollisionPrimitive For the CYLINDER and CONE types, the center line is oriented along the Z axis. Therefore the CYLINDER_HEIGHT (CONE_HEIGHT) component of dimensions gives the height of the cylinder (cone). The CYLINDER_RADIUS (CONE_RADIUS) component of dimensions gives the radius of the base of the cylinder (cone). Cone and cylinder primitives are defined to be circular. The tip of the cone is pointing up, along +Z axis.

The following lists explains the amount of parameters for each collision primitive kind:

  • Plane Representation of a plane, using the plane equation ax + by + cz + d = 0
  • Box Representation of a box
  • Sphere Representation of a sphere
  • Cylinder Representation of a cylinder
  • Cone Representation of a cone
